# VibeCodedComponents

A collection of high-quality, vibe-coded SwiftUI components built with Swift.

## What's inside

Hand-crafted SwiftUI components focused on clean aesthetics and smooth feel — built through vibe coding rather than rigid specification.

## Requirements

- Xcode 15+
- iOS 17+ / macOS 14+
- Swift 5.9+

## Usage

Open `VibeCodedComponents.xcodeproj` in Xcode, browse the components, and copy what you need into your own project.
